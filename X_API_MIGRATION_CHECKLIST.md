## 💰 コスト試算（従量課金の場合）✅ 確定（修正版）

### 実績ベース（月15,000 Posts使用）⚠️
- 生徒数：約150名
- 月間評価回数：2回
- 月間Posts使用量：15,000 Posts（ほぼ上限）
- User情報取得：150〜300回

### 従量課金の料金体系（2026年2月〜）
- **Post読み取り**: $0.005/Post
- **User情報取得**: $0.01/User
- **24時間重複排除**: 同じリソースは24時間以内に複数回取得しても1回のみ課金

### 月間コスト計算
- Post読み取り: 15,000 Posts × $0.005 = **$75.00**
- User情報取得: 150〜300 Users × $0.01 = **$1.50〜$3.00**
- **月間総コスト: $76.50〜$78.00**

### 旧プランとの比較
| プラン | 月額 | 年間節約額 | 削減率 |
|--------|------|----------|--------|
| 旧Basic $100 | $100 | **$264** | **22%削減** |
| 旧Basic $200 | $200 | **$1,482** | **62%削減** |
| 従量課金 | **$76.50〜$78** | - | - |

### 🚨 重要な発見
- **損益分岐点: 20,000 Posts（$100）**
- **現在: 15,000 Posts（$78）**
- **余裕: 5,000 Posts（33%）のみ**

**⚠️ 結論: 従量課金は安いが、学生数が200名を超えると逆転する可能性**
